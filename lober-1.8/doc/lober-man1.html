NAME

lober - Computation of lobes, lobe area, curve intersections and primary intersection points (PIPs).  

SYNOPSIS

lober -h

lober unsfile stbfile lobefile manifile pipsfile unsskip stbskip nZones [ lobefile [ lobeseq ] ]

lober -light separfile lobefile lobeoutfile [ -DENS nPass [ nDens ] ]

 

DESCRIPTION

SYNTAX 1

unsfile  File containing unstable manifold

stbfile  File containing stable manifold

lobefile         File to output colored lobes

manifile         File to output simplified manifolds

pipsfile         File to output pips

unsskip  Zones to skip in unsfile (often 0)

stbskip  Zones to skip in stbfile (often 0)

nZones   Number of zones (often 1)

lobebfile        File to output lobe boundary

lobeseq  Divide lobes in sequences (default=2)

SYNTAX 2 ( -light mode )

separfile        File containing separatrix

lobefile         Files containing lobes

lobeoutfile Files to output cleaned lobes

nPass    See -DENS option below

nDens    See -DENS option below

 

OPTIONS

-h --help        Display syntax and help. This option invalidates any other arguments.

-light Light mode for curves with tangent intersections. Syntax is different for light mode (see SYNTAX 2 above).

-DENS This option is only available for the -light mode (i.e. SYNTAX 2). By default, lober will try to insert points on the curve near their intersections. This is the automatic conditionning (without the -DENS option). The -DENS option allows the insertion of nDens points uniformely distributed on segments of curve that contains an intersection. The operation is performed nPass times. Notice that, by default, there is insertion of 1 point at the intersection point. To avoid the automatic insertion and use the curves as they are, use "-DENS 0". The arguments of -DENS are

        nPass   number of passes (required, can be zero)

        nDens   number of points to add (default=10)
